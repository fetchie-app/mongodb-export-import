# mongodb-export-import

## Installation

### Install Nodejs

Use [NVM](https://github.com/nvm-sh/nvm) to install nodejs

### Install Packages

```bash
cd mongodb-export-import
npm install
```

### Run commands

#### Backup

```bash
npm run start backup <mongodb-url>
```

#### Restore

```bash
npm run start restore <mongodb-url>
```
